README – N’KEMBA UNIVERSAL V1.0

1️⃣ Cria uma pasta chamada nkemba_universal_V1.0
2️⃣ Coloca dentro os ficheiros:
    - index.html
    - servicos-consultoria.html
    - README-INSTRUCOES.txt
3️⃣ Clica com o botão direito → “Enviar para → Pasta ZIP (comprimida)”
4️⃣ Nomeia o ficheiro: nkemba_universal_V1.0.zip
5️⃣ Entra na Domínios.pt → “Gerir Domínio” → nkemba.online
6️⃣ Faz upload do ZIP (ou envia os ficheiros por FTP)
7️⃣ O site fica acessível em https://nkemba.online
